# **App Name**: RideWise Nepal

## Core Features:

- Fuel Log: Record fuel fill-ups with date, odometer, liters, price, and amount.
- Service Log: Maintain a service history including date, odometer, work description, parts, labor costs, and reminders.
- Trip Planner: Plan trips by specifying destination, start date and estimated total distance and daily usage.
- Maintenance Advisor: Based on trip plan, calculate maintenance status. Show alerts/warnings for upcoming or past-due maintenance using generative AI tool.
- Documentation Vault: Keep track of images and document numbers, sorted by category, so it can be shown when requested.

## Style Guidelines:

- Primary color: Crimson (#D10035) for a vibrant and assertive feel, inspired by the Nepal flag.
- Background color: Very light pink (#F8E4E9) to complement the primary, but softened.
- Accent color: Deep purple (#301934) to give richness and depth, and highlight actionable items.
- Body and headline font: 'Inter', sans-serif, known for its modern and readable design.
- Use Lucide React icons for a clean, consistent, and modern look.
- Employ a card-based layout with rounded corners, glass morphism, and subtle shadows for a modern and organized look.
- Use subtle animations like fade-in, slide-in, and scaling on interactions to enhance user experience without being distracting.
